# scripts Folder

## Tujuan
Menyimpan script utilitas untuk testing dan packaging.

## Pencapaian Saat Ini
- `test-lib.ps1` untuk test library.
- `package-plugin.ps1` untuk membuat paket plugin zip.

## Cara Pemakaian
1. Jalankan script via PowerShell dari root project.
2. Contoh: `./scripts/package-plugin.ps1`.
3. Cek output script sebelum lanjut ke tahap berikutnya.
