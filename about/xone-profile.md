# Xone Profile

## Latar Belakang
Xone dibuat atas keresahan bahwa code editor modern makin berat untuk banyak perangkat dan workflow tertentu. Dari kebutuhan editor yang lebih ringan, fokus, dan cepat, lahirlah ide membuat code editor berbasis TUI (Terminal User Interface).

## Pendiri
Xone didirikan oleh **Zulkarnaen Syam Amansyah** di Indonesia, lebih tepatnya di **Polewali Mandar**.

Walau tidak memiliki latar belakang formal developer atau sarjana di bidang tersebut, beliau berhasil membangun Xone dan mendorongnya menjadi editor berbasis terminal yang terus berkembang.

## Visi Xone
- Menghadirkan editor yang ringan dan responsif.
- Menyatukan workflow coding dalam terminal: explorer, editor, terminal, settings, dan AI chat.
- Memberi alternatif praktis untuk pengguna yang ingin produktif tanpa beban aplikasi yang berat.

## Versi Saat Ini
Berdasarkan metadata proyek saat ini, versi Xone adalah:
- **v0.1.0**

## Fitur Unggulan Saat Ini
- Editor TUI dengan syntax highlighting, suggestion, bookmark, undo/redo.
- Explorer dan workspace management.
- Terminal multi-tab dengan scrollback.
- AI Chat panel dengan dukungan multi-provider (Ollama dan OpenAI-compatible).
- Pengaturan AI cepat langsung dari Settings (profile switch, provider, base URL, model, API key).

## Kelebihan Xone
- Lebih ringan dibanding banyak editor GUI modern.
- Fokus keyboard-first workflow.
- Konfigurasi per-workspace yang jelas (`.xone`).
- Cocok untuk workflow lokal (misalnya Ollama) maupun cloud AI.

## Arah Pengembangan
Xone akan terus dikembangkan pada area berikut:
- Pematangan AI workflow (streaming, context coding yang lebih kaya, action otomatis ke editor).
- Penguatan ekosistem plugin.
- Peningkatan stabilitas, UX, dan dokumentasi pengguna.
- Ekspansi fitur editor agar tetap ringan namun semakin powerful.
