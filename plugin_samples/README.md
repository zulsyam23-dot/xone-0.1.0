# plugin_samples Folder

## Tujuan
Menyimpan contoh plugin untuk eksperimen dan validasi alur plugin.

## Pencapaian Saat Ini
- Tersedia plugin contoh `hello-hook`.
- Tersedia output zip di `plugin_samples/dist`.

## Cara Pemakaian
1. Edit plugin contoh di `hello-hook`.
2. Jalankan script packaging.
3. Gunakan file zip untuk uji instal/upload plugin.
